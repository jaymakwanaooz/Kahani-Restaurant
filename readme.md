# Kahani Restaurant

A modern, interactive website for Kahani Restaurant featuring a dark theme, responsive design, storytelling sections, signature dishes, and seamless reservations. Built with Tailwind CSS and custom JavaScript for a premium dining experience.

## Features
- Responsive, mobile-friendly layout
- Story-driven homepage and editorial sections
- Signature dishes and curated menu
- Smooth navigation and scroll animations
- Parallax and interactive effects
- Reservation and contact pages
- Built with Tailwind CSS and custom JS

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/jaymakwanaooz/Kahani-Restaurant.git
   ```
2. Open `index.html` in your browser to view the site locally.

## Project Structure
- `index.html`, `menu.html`, `reservations.html`, `contact.html` – Main pages
- `assets/css/` – Stylesheets
- `assets/js/` – JavaScript for interactivity
- `assets/images/` – Image assets
